Why Murmur

Local. Fast. Private.

Cloud dictation is fast because someone else's GPU is doing the work — and keeping your audio. Murmur runs the same class of model on your Mac, in under a second, and forgets the audio the moment you release the key.

Local

Audio is buffered in memory only while you hold the key, transcribed on-device, and discarded the moment the transcript lands. Nothing is written to disk. Nothing is uploaded.

Fast

Sub-second warm-up on Apple Silicon. MLX-accelerated Whisper Small runs at conversational latency, with Tiny and Base available if you want even less to load.

Private

No accounts, no analytics, no crash reports, no cloud sync. The only outbound request Murmur ever makes is the first-time model download from Hugging Face.

How it works

Three keystrokes you'll forget you're making.

Murmur lives in your menu bar. There's no window to open, no UI to learn. Hold a key, talk, let go.

01

Hold the key

Press and hold Right Option (or Fn on Apple keyboards). A subtle indicator confirms Murmur is listening.

Murmur vs cloud dictation

Same speed. None of the eavesdropping.

The trade-off most dictation apps make is "send us your audio, we'll send you text back." Murmur skips the round-trip — and the data exhaust that comes with it.

Questions

Frequently asked.

Does any audio ever leave my Mac? +

No. The microphone stream is buffered in memory only while you hold the dictation key. It's transcribed on-device by a Whisper model running through MLX, and discarded as soon as the transcript is produced. No raw audio is ever written to disk, and no audio is ever sent over the network.

What model does it use? +

The default is mlx-community/whisper-small.en-mlx-q4 — a quantized, MLX-accelerated version of OpenAI's Whisper Small (English). You can switch to Tiny or Base from the menu bar if you'd rather optimize for speed or memory.

Will it work on Intel Macs? +

Murmur is built on MLX, which targets Apple Silicon. It will not run on Intel Macs. M1 or later required.

Can I change the hotkey? +

Yes. Right Option is the default because it's a low-collision key, but you can rebind to Fn, Caps Lock, or any modifier from the menu-bar settings.

Does it support languages other than English? +

The default model is English-only for speed. Switching to a multilingual Whisper checkpoint (Whisper Small/Base, non-.en) enables 99 languages. The first switch downloads the model; subsequent runs are offline.

Where does my history live? +

Locally, in ~/Library/Application Support/Murmur/history.json. Text only, capped at 500 entries, and clearable from the Home tab. There is no cloud sync, no analytics, no crash reporting — nothing leaves your Mac.

Is Murmur open source? +

Yes — MIT licensed. The source is on GitHub. The privacy promise is auditable: read the network code yourself.
